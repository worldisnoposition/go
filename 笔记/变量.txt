4.const声明一个常量，常量类型只可以是布尔型，数字型（整数，浮点和复数），字符串型
    1)常量还可以做枚举 
        const(
            Unknowed = 0
            Female = 1
            Male = 2
        )
    2)iota 特殊常量，可以认为是可以被编译器修改的常量
        iota再const关键字出现时被重置为0（const内部的第一行之前），const中每新增一行常量声明将使iota计数一次,iota可理解为const语句块的索引
        iota可以被用作枚举值
        const(
            a = iota
            b = iota
            c = iota
        )
        相当于
        const(
            a = 0
            b = 1
            c = 2
        )
        以上代码还可以简写为
        const(
            a = iota
            b
            c
        )
        const (
            a = iota   //0
            b          //1
            c          //2
            d = "ha"   //独立值，iota += 1
            e          //"ha"   iota += 1
            f = 100    //iota +=1
            g          //100  iota +=1
            h = iota   //7,恢复计数
            i          //8
        )
